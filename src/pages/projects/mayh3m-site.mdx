# Introduction

Mayh3m clothing is a clothing brand a friend of mine started. I built this web app to allow users to apply for new mayh3m clothing drops. The app is built with Next.js, React, and trpc. It uses NextAuth for authentication and Prisma for the data access.

# Features

- User authentication
- User profile
- User application
- Admin dashboard
- Admin application management
- Admin user management
